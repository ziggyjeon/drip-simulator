# DRIP 배당 시뮬레이터

세후 실수령 기준 배당 재투자(DRIP) 복리 시뮬레이터입니다.

## 기능

- **세금 정확 계산**: 원천징수 + 종합과세 추가세 + 건강보험료 (소득세법 제47조 근로소득공제 적용)
- **누진세율 자동 계산**: 근로소득과 배당소득 합산 한계세율
- **생활비 전환 시나리오**: 지정 연차부터 DRIP 중단 → 배당금 전액 인출
- **종목 프리셋**: SCHD / JEPI / JEPQ / KB금융 / 신한지주 / 하나금융 / KT&G
- **PWA 지원**: 홈 화면 추가, 오프라인 작동

## 프리셋 근거

| 종목 | 배당수익률 | 배당성장률 | 주가성장률 | 원천세 |
|------|-----------|-----------|-----------|--------|

- 국내 금융지주 주가성장률 3%: 2015~2023 장기 CAGR 2~4% 기준 보수적 추정 (2024~25년 급등은 밸류업 이벤트로 미반영)
- SCHD 배당성장률 7.5%: 10년 CAGR 10.7%이나 최근 3년 7.1% 둔화 반영

## 세금 계산 방식

```
총급여 → 근로소득공제(소득세법 §47) → 근로소득금액 → 기본공제(150만) → 근로소득 과세표준

배당 2,000만 이하: 원천징수로 종결
배당 2,000만 초과:
  종합과세 추가세 = (배당 - 2,000만) × (한계세율 - 원천세율)
  건보료 추가     = (배당 - 2,000만) × 7.09% [직장가입자]
```
